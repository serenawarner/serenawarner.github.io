Senior Independent Study

This Independent Study project uses computational modeling to estimate Jungian cognitive functions from Big Five personality data. Using these proxy estimates, I apply PCA to explore whether the eight cognitive functions reduce to fewer interpretable dimensions. This work investigates how machine learning can be used to revisit and empirically test long-standing psychological theories.

Skills: Python, pandas, NumPy, Matplotlib, scikit-learn, PCA

üé≠ Predicting Myers-Briggs Types with Cognitive Functions and Text Input

Junior Independent Study

This Junior Independent Study project applies supervised machine learning (SVM + Logistic Regression) to classify Myers-Briggs personality types based on forum text. Instead of the four-letter MBTI format, the project uses cognitive function theory (Carl Jung; John Beebe) to model six function-based classification schemes.

Skills: Python, pandas, Matplotlib, NumPy, NLP, SVM, Logistic Regression
